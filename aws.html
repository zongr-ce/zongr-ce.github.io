Automatic Weather Station – Embedded Systems

Overview

Developed embedded firmware for an autonomous Automatic Weather Station designed to operate unattended in Antarctica under strict power and reliability constraints.

My Role

Embedded Systems Engineer responsible for firmware development, power optimization, and system validation.

Technical Highlights

  • Embedded C/C++ firmware using finite state machine architecture
  • Low-power sleep mode implementation
  • Adaptive frequency sampling using timers and ISRs
  • USART and I2C communication protocols
  • Git/GitHub version control (CLI)

Results

  • Reduced idle current from 32 mA to 7 mA (~78% reduction)
  • Reduced idle runtime from 10 minutes to 1 minute 30 seconds
  • System independently operated in Antarctica for a full year
