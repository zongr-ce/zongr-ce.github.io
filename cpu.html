8-Bit CPU – FPGA & PCB Implementation

Overview

Designed and implemented a custom 8-bit CPU from logic gates through FPGA deployment and PCB validation.

My Role

Digital Design Engineer responsible for datapath, control logic, FPGA testing, and hardware validation.

Technical Highlights

  • Von Neumann architecture
  • Custom datapath and control unit
  • Instruction set: FETCH, LOAD, ADD
  • FPGA implementation using Intel Quartus
  • PCB extension for hardware verification

Results

  • Achieved stable operation at 15 kHz after resolving timing violations
  • Verified instruction execution on FPGA hardware
  • Successfully transitioned design from simulation to physical hardware
